Dynamic Control Shapes

Switching control types on the fly

April 18, 2026
toolsuiblender

Control curves are the steering wheel of a rig, and animators are picky about them for good reason. A shape that reads perfectly on a full-body pose becomes clutter on a close-up. This is a small tool with an outsized impact on day-to-day animation: swapping control shapes on the fly, without rebuilding anything.

Context decides the shape

The same control should look different depending on what the animator is doing. Blocking a walk? Big, obvious shapes. Polishing a facial close-up? Thin, unobtrusive ones. The panel lets you switch the entire display set based on context instead of committing to one compromise at build time.

Built as a proper tool panel

This isn't a script you run and forget. It's a UI panel that lives in the rig, exposing the curve library so any animator on the team can change shapes without touching the underlying setup. The control object stays the same; only its visual representation changes.

  • Swap by context — full-body, facial, prop
  • Swap by visibility — declutter dense scenes instantly
  • Swap by preference — animators keep their own muscle memory

Why this matters

Small friction, repeated thousands of times a shot, is what burns animators out. Removing a rebuild step from a daily action is the kind of unglamorous win that a rig actually gets thanked for.
